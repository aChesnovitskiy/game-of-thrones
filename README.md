# Game of Thrones
The knowledge base about the characters of the seven houses of the Game of Thrones

Using [An API of Ice And Fire](https://www.anapioficeandfire.com/)

### Using technologies
* Android Architecture Components
* GSON
* Retrofit
* RxJava
